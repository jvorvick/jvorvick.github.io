Game Picker - Solo Project

Aug 2021 - Sept 2021

This app potentially serves as a database of all known video games and a way to help the user decide which game to play. It is searchable and can be filtered by console and genre. Clicking a button will pull up a random game. A profile can be created by the user. The user will have the option to search for games and add them to their own digital collection. They can then click “randomize” to draw from this collection. The user can also set filters for their own collection before clicking “randomize.” The app will randomize based on this filter.

Tech Skills used: Python, Django, API, JavaScript, HTML, CSS, front-end framework (Materialize)
